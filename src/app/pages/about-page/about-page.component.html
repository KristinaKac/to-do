О проекте

Приложение для планирования и создания списков дел и задач.

Запуск проекта:

  • Для запуска JSON Server необходимо выполнить команду: json-server -w db.json
  • Для запуска frontend необходимо выполнить команду: npm run start

Характеристики проекта:

  • Angular 17
  • OnPush-стратегия
  • Авторизация
  • Библиотеки: Angular HttpClient, Angular Router, Angular Forms
  • Гуард
  • Интерсептор

Функционал списка задач:

  • Отображение списка задач и их статуса выполнения
  • Добавление задач
  • Фильтрация и поиск задач по названию и статусу
  • Изменение статуса задачи
  • Удаление задачи
